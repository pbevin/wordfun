Wordfun API Mode

This server is only serving API requests.

  • In development, the API server normally runs on port 4000, with the React front-end on port 3000. You can run the React front-end with "yarn start".
  • In production, the server is run with --assets-dir=<path>, where <path> points to the build directory produced by "yarn run build".

To try out the API, you can use one of the following paths:

  • /preview/an?q=spear
  • /preview/fw?q=h.r./...l../e.g
  • /preview/thesaurus?q=dog
  • /words/an?q=spear
  • /words/fw?q=a.b.
